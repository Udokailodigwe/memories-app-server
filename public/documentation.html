pet-memories-app-server API Documentation

The documentation on the functionality and understanding of how this API works is detailed below.

pet-memories-app-server REST API goal:

The goal of this REST API is to provide pet lovers opportunity to upload, share and store their memories with their pets. This app enables simple CRUD operations on an application, such create, update, delete, get and like a post.


HTTP Method Request URL Request body data format Response body data format
GET Get posts '/' None Get data array of all posts in JSON format
POST Allow users to create posts '/' A JSON object holding data about the new post
Example: { Creator : 'Pedro', Title: "My cute pet" Message: "Me and cat Selina having fun" Tags: "pet,cat,pedro" } 		A JSON format of the post added information, included with a unique id.
Example: { id: 313abc8798298eetwewe222445, Creator : 'Pedro', Title: "My cute pet" Message: "Me and cat Selina having fun" Tags: "pet,cat,pedro }
PUT Allow user to update their post '/:id' None A JSON format data with the updated result
DELETE Allow users to remove a post from posts '/:id' None Returns empty array
